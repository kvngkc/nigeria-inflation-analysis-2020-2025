# Nigeria Inflation Analysis 2020–2025
### Oil, Naira & the Nigerian Household

## Overview
Analysis of how crude oil prices and the naira exchange rate 
drove inflation in Nigeria between 2020 and 2025, and what it 
cost the average household.

## Key Finding
Nigeria's inflation crisis was not an oil story — it was an 
exchange rate story. When the naira collapsed from ₦306 to 
₦1,670, everything else followed.

## Key Numbers
- Naira depreciation: ₦306 → ₦1,670 (446% decline)
- Peak headline inflation: 34.80% (Dec 2024)
- Peak food inflation: 40.66% (Jun 2024)
- Household basket cost: ₦45,000 → ₦180,503 (301% increase)
